# יהודה שיפוצים — אפליקציית אנדרואיד מתעדכנת

חבילה זו היא אפליקציית ווב מתקדמת (PWA) שהופכת לאפליקציית אנדרואיד אמיתית שמתעדכנת לבד.

## מה יש בחבילה
- `index.html` — האפליקציה
- `sw.js` — Service Worker (עבודה אופליין + עדכון אוטומטי)
- `manifest.webmanifest` — הגדרות האפליקציה והאייקונים
- `icon-192.png` / `icon-512.png` / `icon-180.png` — אייקונים
- `version.json` — מספר גרסה לבדיקת עדכונים

---

## שלב 1 — פרסום לאינטרנט (Netlify · חינם)
1. היכנס ל-app.netlify.com → Add new site → Deploy manually.
2. גרור את **כל התיקייה הזו** (או קובץ ZIP שלה) לאזור ההעלאה.
3. קיבלת כתובת כמו `https://yehuda-shipuzim.netlify.app`.
   (אפשר לשנות שם ב-Site settings → Change site name.)

מרגע זה האפליקציה חיה. אפשר כבר "הוספה למסך הבית" מכרום והיא תעבוד כאפליקציה.

## שלב 2 — יצירת קובץ APK (PWABuilder · חינם)
1. היכנס ל-**pwabuilder.com**.
2. הדבק את כתובת ה-Netlify שלך → Start.
3. לחץ **Package For Stores** → **Android** → **Generate**.
4. הורדת קובץ `.apk` (להתקנה ישירה) ו-`.aab` (לחנות Play).
5. התקן את ה-APK בטלפון, או העלה את ה-AAB ל-Google Play Console.

## שלב 3 — עדכונים (זה החלק שמתעדכן לבד)
- ה-APK הוא עטיפה שטוענת את האתר החי שלך. **כל שינוי שתעלה ל-Netlify מופיע אוטומטית באפליקציה** בפתיחה הבאה — בלי להתקין מחדש.
- כשאני נותן לך גרסה חדשה: החלף את `index.html` ב-Netlify, ועדכן את מספר הגרסה בשני מקומות:
  1. בתחילת `sw.js` בשורה `const CACHE = 'ys-cache-vX';`
  2. בקובץ `version.json`
  ה-Service Worker יזהה, יתקין את הגרסה החדשה ברקע, וירענן את האפליקציה לבד.

## הגדרות אחרי התקנה (בתוך האפליקציה)
- **⚙️ הגדרות → GREEN-API**: הזן apiUrl, idInstance וטוקן → שליחת וואטסאפ שקטה (עותק אליך + PDF ומשוב ללקוח).
- **⚙️ הגדרות → Google Drive**: הזן Google OAuth Client ID → תיקיות תמונות "לפני/אחרי" לכל לקוח + גיבוי בענן.

בהצלחה! 🛠️
